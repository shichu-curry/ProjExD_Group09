# Swarm Night
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* Vampire Survivorsのように見下ろし型のマップにおいて、四方から近づいてくる敵の群れから生き残るサバイバルゲーム。
* 主人公（こうかとん）は自動で攻撃を行うため、プレイヤーは移動による立ち回りに集中してできるだけ長く生存し、撃破数を稼ぐことを目指す。
* 参考URL：[Vampire Survivors](https://game-ac.com/free/gunshooting/vampire-survivors/)

## ゲームの遊び方
* 矢印キーまたはWASDキーで主人公を8方向に操作する
* 攻撃は自動で行われ、一定間隔で最も近い敵に向かって弾が発射される
* 敵に接触するとHPが減少し、HPが0になるとゲームオーバーとなる
* ゲームオーバー画面でRキーを押すとリスタート、ESCキーで終了する
* 弾に加えて、剣（周囲を一回転する斬撃）・槍（向いている方向への突き）・斧（放物線を描く投擲）の3種類の武器も一定間隔で自動発動する

## ゲームの実装
### 共通基本機能
* メインループとゲーム状態管理（PLAY／GAMEOVERの状態遷移）
* 主人公キャラクターの描画と8方向移動（HP・被弾時の無敵時間つき）
* 敵キャラクターの描画と画面外からのスポーン・追尾移動
* 最も近い敵へ自動発射する武器と弾の描画・移動
* 弾×敵、敵×主人公の当たり判定とダメージ処理
* HPバー・経過時間・撃破数を表示するHUD
* ゲームオーバー画面の表示とリスタート処理

### 分担追加機能
* ノックバック機能（担当：C0C25014　大島澪）：敵との接触時に、敵と反対方向へ一定距離吹き飛ばされる機能を実装。
* スタミナ制限付きダッシュ機能（担当：山内蓮）：移動中にShiftキーを長押しすることで、通常の1.6倍の速度で高速移動ができる機能。* ダッシュ中はスタミナを消費し、0%になると通常速度に戻る。非ダッシュ時・停止時にはスタミナが自動で徐々に回復する。画面左上のHUD（HPバーの下）にオレンジ色のスタミナゲージを上書き描画して管理。
* アイテムドロップ機能（担当：村山）：敵撃破時に回復アイテムや経験値ジェムをドロップし、取得できる機能
* 敵の撃破時に経験値を確定でドロップ。また、一定確率でHPを20回復できる回復アイテムを敵がドロップする。
* 難易度上昇機能（担当：牧田）：
* 武器の種類追加（担当：杉本）：今回は、既存の弾攻撃以外の攻撃手段として、剣・槍・斧の3武器を実装した。
  「ゲームの遊び方」に、弾に加えて剣・槍・斧の3種類の武器が一定間隔で自動発動することを1行追加した。
    実装した各種武器に関しては、剣（プレイヤーの周囲を一回転する斬撃）・槍（向いている方向へ突き出す貫通攻撃）・斧（放物線を描いて飛ぶ投擲）の3武器を実装した。
    攻撃はattacksグループで一括管理され、hit_idsによって同じ敵には1回だけ当たる貫通型の当たり判定を持つ。
    武器画像はfig/フォルダのpng（sord01.png・yari01.png・ono.png）をload_image関数で読み込み、画像が無い環境でも簡易図形で代用して動作する。


### メモ
* 主人公・敵・弾はすべてpygame.sprite.Spriteを継承し、Groupで一括update／drawするように設計してある
* 敵はEnemyクラスを親クラスとして継承で種類を追加し、武器は「クールダウンが切れたら発動する」共通の形式に揃えてある
* 敵の湧き間隔・HP・弾速などのバランス調整用の数値は、すべてファイル冒頭の定数にまとめてある
* 1ファイルの変更で1コミットをして、大量の行数の変更差分を一度に行わないように、細かくコミットすることを心がる。
